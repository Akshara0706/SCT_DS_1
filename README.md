# SCT_DS_1 - Population Distribution Visualization

## Overview

This project analyzes population data using Python and visualizes key patterns through histograms and bar charts.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Visualizations

1. Population Distribution Histogram
2. Gender Distribution Bar Chart
3. Region Distribution Bar Chart

## Project Objective

Create visual representations of categorical and continuous variables to identify trends and distributions within the dataset.
